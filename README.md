# proyecto-tecnologicolagunade-tesorero
Este proyecto se basa en conocer un poco más la historia de la sede de mi escuela y mostrar su recorrido comunitario, cultural y tecnológico.

##  Contenido del proyecto
- **entregar.html** → Página principal del proyecto.
- **estilos.css** → Diseño visual del sitio.
- **archivo.ccl** → Archivo de apoyo del proyecto (código/trabajo).
- **LICENSE** → Licencia MIT.
- **README.md** → Explicación del proyecto.

##  Objetivo
Investigar, organizar y presentar información relevante sobre la sede Laguna de Tesorero, resaltando su identidad comunitaria y educativa.

##  Tecnologías utilizadas
- HTML  
- CSS  
- Documentación digital  
- Herramientas de edición escolar
